🛠️ Technical Details

The implementation consists of three components:

  • JavaScript Viewer (route: /oai) - Fetches and transforms XML to styled HTML
  • CSS Stylesheet (priv/static/css/oai-pmh.css) - Provides visual styling
  • OAI-PMH API (route: /api/oai) - Returns standard OAI-PMH XML responses

The viewer uses modern JavaScript to:

1. Fetch XML from /api/oai endpoint 2. Parse XML with DOMParser 3. Transform to HTML dynamically 4. Apply CSS styling for beauty!

💡 Access Raw XML

Need the raw XML for harvesting or debugging? Access it directly:

  • Direct API: http://localhost:4000/api/oai?verb=Identify
  • Command line: curl "http://localhost:4000/api/oai?verb=Identify"
  • In viewer: Open browser DevTools → Network tab → Find the /api/oai request

The raw XML is always available at /api/oai - harvesters can use it directly without going through the viewer.
